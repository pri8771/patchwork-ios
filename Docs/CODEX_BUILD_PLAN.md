2. Swift package foundation+tests
3. core models+visited bitset
4. geometry+point-in-polygon
5. spatial index
6. weighted rollup engine
7. offline Census Python pipeline skeleton
8. runtime geodata repository+sample dataset
9. SwiftData persistence
10. MapKit bridge overlay prototype
11. location capture + Claim Current Patch
12. core UI shell from Claude design
13. share card generation
14. StoreKit 2 monetization
15. export/import/reset/privacy
16. performance guardrails
17. full geodata integration
18. final hardening+release readiness
